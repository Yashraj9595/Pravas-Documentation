Common Navigation System

This page demonstrates the common navigation system. The navigation should be loaded dynamically from the [navigation.html](file://c:\Users\yashr\Downloads\sih\documentetion%20web\Pravas-Documentation\navigation.html) file.

Features:

  • Click-based Dropdowns: Click on the dropdown icons to open/close menus
  • Active Page Highlighting: Current page is highlighted in the navigation
  • Common Navigation: All pages now use the same navigation structure
  • Responsive Design: Navigation works on both desktop and mobile

How it works:

  1. Each page has an empty sidebar container
  2. JavaScript loads the common navigation from [navigation.html](file://c:\Users\yashr\Downloads\sih\documentetion%20web\Pravas-Documentation\navigation.html)
  3. Links are adjusted based on page location (root vs subdirectory)
  4. Active page is highlighted automatically
  5. Dropdowns open/close on click, not hover
